Sobre MyFindings

Motivación

Hoy en día, para la toma de datos durante las excavaciones se usa un procedimiento tradicional, con lápiz y papel, lo cuál dificulta en gran medida el trabajo. Frente a este problema, surgió MyFindings, una aplicación que te permitirá realizar una toma y gestión de los datos de una manera increíblemente fácil e intuitiva. Características principales:

  • Gestión de excavaciones
  • Sincronización en tiempo real
  • Añade, edita, elimina excavaciones
  • Generación de informes
  • Rápido, eficiente, intuitivo
